# 📝 Notes App

## 📌 Description

This is a simple Notes App built using Python.
It allows users to write, save, and view notes easily.

## 🚀 Features

* Add new notes
* View saved notes
* Simple and user-friendly
* Runs in terminal

## 🛠️ Tech Used

* Python

## ▶️ How to Run

1. Download the project files
2. Open in VS Code or any Python editor
3. Run the main file
4. Follow the instructions on screen

## 💡 Future Improvements

* Add delete/edit notes feature
* Create a GUI version
* Store notes in a database
